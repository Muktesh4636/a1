## üéØ Game Overview
Gundu Ata is a game where players bet on numbers (1-6). In each round, **six dice** are rolled, and winning numbers are determined based on their frequency.

## üèÜ Winning Criteria
A number is considered a **winner** if it appears **2 or more times** in a single roll of the six dice.

- **Multiple Winners:** It is possible for more than one number to win in a single round (e.g., if the roll is `4, 4, 4, 2, 2, 1`, both **4** and **2** are winning numbers).
- **Most Common Winner:** For display purposes, the number with the highest frequency is shown as the primary winner. In case of a tie in frequency, the first occurring number in the roll sequence is chosen.

Payouts are calculated based on the frequency of the winning number.

### 2. No Commission (100% Payout)
There are **no deductions** or commissions. The player receives **100%** of the calculated payout directly into their wallet.

**Example Calculation:**
- Player bets **‚Çπ100** on number **4**.
- Number **4** appears **3 times**.
- **Total Payout:** ‚Çπ100 √ó 3 = **‚Çπ300**.
- **Player Receives:** **‚Çπ300** (Full amount credited to wallet).

## ‚è±Ô∏è Game Timing (Default)
Each round follows a strict timeline (configurable in Game Settings):

1.  **Betting Open (0s - 30s):** Players can place, increase, or remove bets.
2.  **Betting Closed (30s - 51s):** No more bets allowed. The system prepares the dice roll.
3.  **Result Announced (51s):** The dice values are revealed, and winners are determined.
4.  **Round Completion (80s):** Payouts are processed, and a new round begins immediately.
